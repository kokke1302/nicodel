# nicodel

ニコニコ動画の『フォロー新着』で、特定の投稿者の動画を非表示にするブラウザ拡張機能。

## 機能

- 非表示にする投稿者の登録・削除・一括リセット
- 『フォロー新着』ページにて、登録した投稿者の動画タイルを自動的に非表示にする

## 制作背景

ニコニコ動画の新着には、フォロー中の投稿者の動画がすべて表示される。
サイバー攻撃以前は、新着に表示しない投稿者を個別に指定できたが、現在はその機能がない。
毎日投稿の投稿者はまとめて週末に観たいなど、新着から除外したい投稿者を手元で管理できるようにするため、拡張機能として自作した。

## 使い方

### インストール

1. このリポジトリを `Clone`、またはZIPでダウンロードして展開する
2. ブラウザの拡張機能管理ページを開き、「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」から `src/` フォルダを選択する

### 投稿者の登録

1. 『フォロー新着』で非表示にしたい投稿者名（太字部分）をコピーする
2. 拡張機能のオプションページを開き、入力欄に貼り付けて「追加」を押す
3. 『フォロー新着』を再読み込みして確認する

### 登録の解除

- オプションページで投稿者名の横の「×」ボタンを押す
- 「すべてリセット」ボタンで一括削除も可能

## 動作環境

Vivaldi（Linux / deb版）（Windows / 公式インストーラー版）、Brave（Linux / deb版）で動作確認済み。
Chromium系ブラウザであれば動くはずであるが、他環境での動作は未確認。

## 仕様

新着25件のうち非表示対象が20件あった場合、表示されるのは残り5件のみとなる。
つまり、ページネーションの補充はない。

## トラブルシューティング

1. 非表示にならない: ページを再読み込みする
2. 名前が認識できてない可能性: オプションページで一度リセットし、登録し直す
3. リスト表示が変: 拡張機能を削除し、再インストール
4. 上記で直らない: 自分でコードを編集するか、Issuesに報告するか、諦める
